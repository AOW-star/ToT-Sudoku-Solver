# ToT思维树数独求解器
## 第1题：ToT适用场景
ToT（Tree of Thoughts，思维树）是大语言模型推理增强技术，核心是多路径探索+评估剪枝+回溯，适合以下场景：

### ✅ 适合用ToT的核心场景
1.  **多步推理/复杂决策类任务**：数学难题、逻辑谜题（数独、华容道）、路径规划，单路径思维链易卡壳，ToT树状探索提升成功率
2.  **约束满足类问题**：会议安排、排班系统、资源分配，多约束下生成候选方案并筛选最优解
3.  **创意/多方案生成类任务**：文案创作、产品方案设计，多方向思路生成+择优
4.  **前瞻/试错的策略类任务**：博弈游戏、项目排期、故障排查，模拟不同分支结果选最优
5.  **复杂问题拆解与分步验证**：代码调试、系统设计，拆分子步骤+验证可行性

### ❌ 不适合用ToT的场景
- 简单问答、事实性查询（单路径足够，ToT增加开销）
- 实时性要求极高的任务（多分支生成耗时）
- 无明确评估标准的开放型任务（难以剪枝）

## 项目说明
本项目用Python实现ToT（思维树）算法，完成数独求解，严格遵循ToT四大核心流程：生成→评估→搜索→回溯。

### ToT实现方法说明
#### 核心流程
1.  **任务分解**：将数独求解拆解为「逐个填充空单元格」的分步任务，每步对应思维树的一个节点
2.  **思维生成**：在每个空单元格，生成1-9的所有候选填数，作为思维树的子分支
3.  **评估与剪枝**：通过`is_valid`方法检查候选数的合法性（行/列/宫格无重复），剪枝无效分支
4.  **搜索与回溯**：用深度优先搜索(DFS)遍历思维树，死胡同自动回溯，找到终态返回解

#### 核心类
- `ToTNode`：思维树节点，存储当前盘面、父节点、子节点，实现生成、评估逻辑
- `ToTSolver`：ToT求解器，用DFS遍历思维树，完成回溯搜索
